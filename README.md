# jonathanwhitmore.com

Personal website and blog built with [Quarto](https://quarto.org/).

## Local Development

```bash
quarto preview
```

To preview posts that use Python kernels, point Quarto at the project venv:

```bash
QUARTO_PYTHON=.venv/bin/python quarto preview
```
